# FastAPI Starter Service

A lightweight REST API, built using FastAPI, to demonstrate a functional Python API with automatic interactive documentation in 20 lines of code.


## Features

- Built with **FastAPI**
- Includes a working **GET endpoint**
- Returns a structured **JSON response**
- Configured with **CORS middleware**
- Runs locally using **Uvicorn**
- Includes automatic **Swagger UI / OpenAPI documentation**
- Minimal, beginner-friendly backend project structure

  
## Tech Stack

**Language:** Python
**Framework:** FastAPI
**ASGI Server:** Uvicorn
**Middleware:** CORSMiddleware
**API Documentation:** Swagger UI / OpenAPI
**Version Control:** Git + GitHub
